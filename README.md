# Stunnel Server Setup

Автоматическая установка и настройка stunnel-сервера на Ubuntu 24.04.

### ⚠️ Важное замечание о безопасности
Скрипт создаёт **самоподписанный сертификат**, что удобно для тестирования. Однако для использования в production-среде настоятельно рекомендую настроить сертификат от Let's Encrypt, чтобы избежать предупреждений безопасности.
